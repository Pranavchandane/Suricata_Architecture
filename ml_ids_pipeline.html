01
NETWORK Network Interface
Raw packets arrive at the network adapter. Npcap (Windows) intercepts all incoming and outgoing traffic on the wire before it reaches the OS networking stack.
Npcap (Windows) libpcap (Linux) AF_PACKET promiscuous mode
→ capture.py
02
CAPTURE Packet Capture
Scapy's sniff() reads raw packets from the interface in real-time. Each packet is passed to a callback function instantly — store=False keeps memory lean.
scapy sniff() store=False prn callback non-blocking
→ capture.py :: PacketCapture.start()
03
DECODE Ethernet → IP → TCP/UDP
Each raw packet is dissected layer by layer. Headers are parsed and stored in a DecodedPacket struct: source/destination IPs, ports, protocol type, TCP flags, and payload size.
Layer 2: Ethernet (MAC) Layer 3: IPv4/IPv6 Layer 4: TCP / UDP / ICMP TCP flags: SYN FIN RST ACK
→ decode.py :: decode_packet()
04
FLOW ENGINE Group Packets Into Flows
Packets are grouped into bidirectional flows using a 5-tuple key. A background thread expires flows after timeout (60s) or on TCP FIN/RST. Expired flows are sent downstream for analysis.
5-tuple: src_ip, src_port, dst_ip, dst_port, proto bidirectional hash key FIN/RST = instant expire timeout = 60s background expiry thread
→ flow_engine.py :: FlowEngine
05
KEY LAYER Feature Extraction
When a flow expires, 24 statistical features are computed from its packets. These numbers represent the behavior signature of the flow — what the ML model actually sees.
flow duration total packets & bytes fwd/bwd packet counts packet length mean/std/min/max inter-arrival times (IAT) bytes/sec & packets/sec SYN / FIN / RST counts src & dst port
→ feature_extractor.py :: FeatureExtractor.extract()
06
ML INFERENCE Model Prediction
The 24-feature vector is fed into a Random Forest classifier trained on CIC-IDS2017. The model outputs a label and confidence score. This replaces Suricata's entire rule engine.
input: [24 features] model: RandomForest (100 trees) output: BENIGN / ATTACK confidence: 0–100% trained on CIC-IDS2017
→ ml_model.py :: MLModel.predict()
07
OUTPUT Alert / JSON Log
If the model predicts ATTACK, a structured JSON alert is written to alerts.json and printed to console. BENIGN flows are silently discarded.
timestamp src_ip : src_port dst_ip : dst_port protocol confidence % alerts.json
→ alert.py :: AlertEngine.generate()
